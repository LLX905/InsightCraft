# InsightCraft

InsightCraft is an AI-powered analytical assistant designed to help users select the most effective data visualizations and explore complex problems through structured mind maps.

## Features

### Visualization Recommendation Engine
Users input:
- Visualization tool (Excel, Tableau, PowerBI, Python, R)
- Data columns or fields
- Data types (numerical, categorical, geographic, time-series)
- Visualization purpose

The engine then recommends:
- The most suitable chart types
- Clear reasoning for the recommendation
- Field mapping instructions
- Tool-specific implementation guidance

### Mind Map Problem-Solving Engine
Users enter a problem or analytical question.

The system generates a structured multi-perspective mind map that:
- Breaks down the problem into root causes
- Explores overlooked perspectives
- Suggests actionable strategies

Mind maps can be exported as:
- PNG

Users can choose:
- Horizontal layout
- Vertical layout

## Tech Stack

- Firebase Studio
- React
- AI-powered reasoning engine
- PNG rendering for mind maps
