# **App Name**: InsightCraft AI – Intelligent Visualization & Analytical Thinking Assistant

## Core Features:

- Dataset Ingestion: Allow users to upload CSV/Excel files, link Google Sheets, or manually define column schema with types and example values.
- AI Data Profiling: An AI tool that automatically detects categorical, numerical, time series, hierarchical, and geographic variables from ingested datasets.
- AI Visualization Recommendation Engine: An AI tool that recommends suitable visualization types based on dataset characteristics and user intent, explaining their suitability and providing configuration guidance.
- Interactive Visualization Display: Render and display interactive data visualizations, allowing users to explore charts recommended by the AI or constructed from their data.
- AI Problem-Solving Mind Map Generator: An AI tool that automatically generates a deep, structured, multi-level mind map analysis from a user-input problem statement. The mind map presents a hierarchical structure with multiple analytical layers (Core Problem, Major Perspectives, Sub-causes, Deeper Root Causes, Potential Solutions) and numerous branches, providing a comprehensive, multi-perspective understanding without user editing. It ensures nodes do not overlap, text is fully visible, and automatically splits into multiple pages for very large diagrams. Users can export the static, analytical mind map as an image (PNG) or PDF.

## Style Guidelines:

- A light color scheme conveying intelligence and professionalism. Primary: A refined, deep blue (#3880E0) to symbolize trustworthiness and analytical depth. Background: A very light, cool grey-blue (#E6EDF6) to provide a clean and calm canvas for data. Accent: A vibrant, clear cyan (#4FC7DB) for highlighting key insights and interactive elements, providing a refreshing contrast.
- Mind Map Node Colors: Problem node as large blue rectangles, Category nodes as medium grey rectangles, Cause nodes as white rectangles, and Solution nodes as green rectangles.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-forward aesthetic. Body text font: 'Inter' (sans-serif) for optimal legibility and neutrality across data-heavy interfaces.
- Utilize a consistent set of clean, minimalist line-based icons. Focus on clarity and instant recognition to guide users through complex data and analysis features.
- Implement a spacious, organized dashboard-style layout, similar to leading analytics platforms like Tableau. Clear module segmentation with distinct areas for data input, visualization canvas, and mind map generation.
- For mind maps, nodes should be presented as square or rounded boxes with clear, connecting lines showing hierarchy (parent → child). The layout algorithm must prevent node overlap, ensure text is fully visible, and maintain sufficient spacing. Users can choose between horizontal or vertical hierarchical structures. Large diagrams will automatically split into multiple pages.
- Subtle, smooth animations for transitions between data views, loading states, and interactive chart elements to enhance user experience without distraction or unnecessary embellishment.
